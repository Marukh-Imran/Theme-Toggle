Website Page with Theme Toggle

A simple, responsive, and beautiful portfolio website template. The main feature of this project is a dynamic JavaScript theme toggle that lets users switch smoothly between Dark Mode and Light Mode.

## 🌟 Features
* **Dark & Light Mode:** Switch themes easily with a single button click.
* **Fully Responsive:** Beautiful layout that looks great on laptops, tablets, and mobile screens.
* **Clean Sections:** Includes clean layouts for About Me, Skills, Projects, and a Contact Form.

## 🛠️ Tools & Technologies Used
* **HTML5:** For structured and clean web content.
* **CSS3:** For modern styling and smooth layout design.
* **JavaScript (JS):** To handle the theme-switching logic.

## 💡 How it Works
1. When you open the website, it loads the standard theme.
2. Clicking the **"Light Mode" / "Dark Mode"** button instantly changes the background and text colors using JavaScript.
